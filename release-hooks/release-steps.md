Release Steps:

1. Verify Conditions
    - Verify if the release should happen.
2. Analyze Commits
    - Determine the type of release.
3. Verify Release
    - Verifying a release that was determined before and is about to be published.
4. Generate Notes
    - Generate the release note.
5. Prepare
    - Prepare the release.
6. Publish
    - Publish the release.
7. Success or Fail
    - Notify of a new/failed release.
